## 정수 N개의 합
### *
> main 필요없이 그냥 함수만 작성하면됨

>몰라서 구글링함
>근데 main작성했는데 stack overflow남..
> '''c++
#include <stdio.h>

long long sum(long long *a, int n) {
	long long ans = 0;
    for(long long i=0;i<n;i++){
        ans+=a[i];
    }
	return ans;
}
int main(){
    int n;
    scanf("%d\n",&n);
    long long *score[3000000];
    printf("%llu",sum(score,n));
    return 0;
}'''
